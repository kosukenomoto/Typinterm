# Typinterm
cli typing trainer

## やりたいこと
 - terminalでタイピング
 - 例文１行　入力１行　情報1行の３行構成
 - 全体行数
 - 入力１字単位で正誤評価
 - WPMの計測
 - 正誤評価結果のリアルタイム表示
 - リソース指定
 - typo率の計測
 - 過去実績との競争
